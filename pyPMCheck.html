Priming Effects Checker (cross-line)

pmcheck function

a python function that checks in materials in an xlsx, if there are the following priming effects:

1. CL in last sentence

2. N in last sentence

3. CL in last probe question

4. N in last probe question

5. four successive conditions

Priming Effects Checker (cross-line) script

do pmcheck to all xlsxs in a folder (and all its subfolders)

how-tos

1. set column number of condition, CL, N, Probe Question, and filler's condition, and SENTENCE_START_COL_NUM, SENTENCE_END_COL_NUM

2. run the script, input folder path and press enter key

3. review the results produced
